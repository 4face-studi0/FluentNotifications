start

inline fun <reified T : Any> start(requestCode: Int? = null, flags: Int = DEFAULT_FLAGS, intentFlags: Int? = null, options: Bundle? = null, isForeground: Boolean = false): Unit

Create a PendingIntent for start a Component of type T

Parameters

T - generic type of the Component to start, bound types are the followings:

requestCode - OPTIONAL Int or IntegerRes for the PendingIntent Default is null

flags - OPTIONAL Int flags for the PendingIntent Default is DEFAULT_FLAGS

intentFlags - OPTIONAL Int flags for the Intent created with the given T type Default is null

options - OPTIONAL Bundle for the PendingIntent NOTE: This param will be used only if T is subtype of FragmentActivity, otherwise it will be ignored Default is null

isForeground - OPTIONAL Boolean whether the Service must be started in foreground.
